Primer trabalho GIT para entregar a Code.education
==================================================

1 - Criei uma pasta com o comando ***mkdir trabalho-git***

2 - Mudei para a pasta trabalho-git com o comando ***cd trabalho-git***

3 - Inicializei o repositorio com o comando ***git init***

4 - Criei o arquivo Readme.txt com o comando ***vim Readme.txt***

5 - Fiz uma ***git add Readme.txt*** para colocar o arquivo em estado de Chages

6 - Comitei o arquivo com o comando ***git commit -m 'Primeiro commit para presentar a Code.education***

7 - Entrei na minha conta github e criei o repositorio em branco ***trabalho-git***

8 - No meu computador Executei o comando ***git remote add origin https://github.com/Danicarna/trabalho-git.git*** para inicializar o repositorio remoto

9 - Para subit ou empurrar os arquivos do repositorio local para o repositorio remoto utilizei o comando ***git push origin master***

10 - Atualizei o ***github*** para verificar se os arquivos tinhas subidos.


Espero ter conseguido os objetivos soliciados.
